## Less, but better

Every year I get more convinced that the best interfaces are the ones that disappear. In 2026, with AI generating
endless variations of everything, the scarce resource is **taste** — the courage to delete.

When I start a project, my first pass is always additive. I throw colors, gradients, motion, copy — anything I can
think of. The real work begins on the second pass, when I start subtracting.

### A short checklist I use

- Could this section be a single sentence?
- Does this animation help the user, or just impress them?
- If I removed this color entirely, would anyone notice?
- Can this two-line label become one word?

### What stays

What survives the cut tends to be:

1. **Type.** A confident typographic system carries 80% of the personality.
2. **Whitespace.** Real, generous whitespace, not "design padding".
3. **One brave move.** A bold color, an unexpected layout, a single moment of motion.

That's it. The rest is noise.
